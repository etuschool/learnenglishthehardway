## train_num.py

---

使用: `train_num <list_path>`

功能: 训练用户掌握词汇表里的所有数字

词汇表格式:

(1) 数字

	2017
	10,000
	98,011,222
	Mon # 星期一
	Feb # 二月份
	1/3 # 三分之一

训练方式:

(1) 数字

	假设需要训练的单词为: 2017
    >> (系统朗读英文数字), 请输入你听到的数字, 输 # 重听; 输 ? 获得更多提示.
    >> 2070
    >> 错误 (再读一遍)
    >> 2077
    >> 错误, 提示: [ two thousand and seventeen ]
    >> 2017
    >> 棒棒哒！下一个～

__提示__: 先尝试将 2017 转成 two thousand and seventeen, 然后用 mac 的系统命令 `say` 读出来。
